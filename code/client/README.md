## Pre-requisites 

#### Modules to install

`npm install --save-dev @angular-devkit/build-angular`

##### Librairies *stompjs* et *socksjs* : 

`npm install stompjs --save`

`npm install sockjs-client --save`

##### Librairies *jquery* et *bootstrap* : 

`npm install jquery --save`

`npm install bootstrap`

## Run client : 

To run 2 different clients:
In */client* folder run commands: 

1st client : `ng serve --open `

2ng client : `ng serve --open --port 4210`
